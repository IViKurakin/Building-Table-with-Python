# Building-Table-with-Python
Создание простых таблиц в Python с использованием библиотеки prettytable | Настройка отображения таблицы: выравнивание столбцов, сортировку по столбцу, добавление заголовков. Импорт данных из CSV-файлов, HTML-кода, SQL-баз данных.

# Необходимые библиотеки для реализации проекта:
1) prettytable
   Для установки через терминал или командную строку используйте: pip install prettytable
   Для установки в среду Google Collab используйте: !pip install prettytable
   Поддержка библиотеки: https://ptable.readthedocs.io/en/latest/tutorial.html
